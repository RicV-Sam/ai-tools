AI Tool Radar is curated manually. If there is a tool you think deserves coverage, you can suggest it by email for future review consideration.

How suggestions work

Suggestions are reviewed manually for relevance, category fit, search value, and whether the tool would improve the usefulness of the site for readers making practical decisions.

How to submit a tool

For now, suggestions should be sent by email manually. Include the tool name, homepage URL, what category you think it belongs in, and why it would be useful for readers.

What to expect

A suggestion does not guarantee inclusion. Tools may be skipped if they are too early, too unclear, not relevant to the site structure, or already well-covered through stronger alternatives.

Editorial alignment

Before sending a suggestion, it helps to read the How We Review AI Tools page so you can see the standards used when deciding what gets added.
